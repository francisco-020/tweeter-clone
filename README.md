# Tweeter Replica (Project 1)

Live site: https://tweeterwebdev3342.netlify.app/
Repo: <https://github.com/francisco-020/tweeter-clone.git>

## What it is
A responsive front-end replica of “Tweeter” built with HTML, CSS, and JavaScript.

## Features
- Character counter (160 max)
- Tweet button disables when limit exceeded
- Responsive layout (mobile → desktop)

## Tech
HTML / CSS / JavaScript

## How to run locally
1. Download or clone the repo
2. Open `index.html` in a browser
